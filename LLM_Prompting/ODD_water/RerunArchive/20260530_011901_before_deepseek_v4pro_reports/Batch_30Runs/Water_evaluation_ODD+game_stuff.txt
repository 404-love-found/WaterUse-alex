TWO-GT EVALUATION: TP / FN / FP (ODD+game_stuff)
==============================================================================

Correct action situations (2):
  AS1: Upstream and downstream withdrawal decisions
  AS2: Fish extraction common pool resource game

Metrics:
  TP = How many LLM-generated ASs were in the correct set of ASs
  FN = How many correct ASs the LLM missed
  FP = How many LLM-generated ASs were not in the correct set of ASs
  Precision = TP / (TP + FP)
  Recall    = TP / (TP + FN)

Scoring rules:
  Each generated AS is reviewed against only the two confirmed correct ASs.
  The first generated AS matching a correct AS counts as TP.
  Additional generated ASs matching the same correct AS count as FP-duplicate.
  Generated ASs matching neither correct AS count as FP-wrong.
  Missing correct ASs count as FN, so TP + FN = 2 for every run.
  Generated AS titles are included with body/matrix text when judging correctness.
