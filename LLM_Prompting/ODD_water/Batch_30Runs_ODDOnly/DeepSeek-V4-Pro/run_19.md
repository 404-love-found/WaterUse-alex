But since the model is dynamic, the tension is between catching more now and preserving the stock for future. In a normal form game, we can represent the long-term payoff as the net present value of catches. If they catch too much, the stock collapses and future catches are zero. So we can set up a game where each player chooses a catch level (High or Low). The sustainable yield is a total catch threshold. If total catch > threshold, the stock collapses and both get zero in the long run (or very low). If total catch <= threshold, the stock persists and they get their catch.

Additionally, the access order matters: Downstream gets first access. So if he chooses High, he might get his high catch, but Upstream might get nothing or very little. So the payoffs can reflect that.
