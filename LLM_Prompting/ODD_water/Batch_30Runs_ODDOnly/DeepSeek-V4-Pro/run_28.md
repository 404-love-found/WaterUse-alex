Payoffs as before: agri net = water_received - 0.2*fields. Fishing: if collapse (total>10), stock=3; else stock=10. Fishing income: D gets min(5, stock), U gets min(5, stock - D's catch). Actually, we need to model the sequential fishing: D fishes first. So if stock=3, D gets 3, U gets 0. If stock=10, both get 5. So for W=15:
U=H(10), D=H(10): U gets 10 water, D gets 5. Agri: U=10-2=8, D=5-2=3. Total extraction=15>10, collapse. Fishing: D=3, U=0. Payoffs: (8,6).
U=H, D=L(5): U=10, D=5. Agri: U=8, D=5-1=4. Total=15>10, collapse. Fishing: D=3, U=0. Payoffs: (8,7).
U=L(5), D=H(10): U=5, D=10. Agri: U=5-1=4, D=10-2=8. Total=15>10, collapse. Fishing: D=3, U=0. Payoffs: (4,11).
U=L, D=L: U=5, D=5. Agri: U=4, D=4. Total=10, no collapse. Fishing: both 5. Payoffs: (9,9).

Matrix Dry:
        D: H      D: L
U: H   (8,6)    (8,7)
U: L   (4,11)   (9,9)

Nash: U dominant H, D best response to H is L (7>6). So (H,L) is equilibrium with payoffs (8,7). This is worse for both than (L,L) (9,9). So the tension: Upstream's dominant strategy leads to a suboptimal outcome, but Downstream cannot prevent it. This reflects the spatial asymmetry: Upstream's water priority allows him to unilaterally cause collapse, and Downstream's fishing advantage doesn't help because collapse destroys fish for both (though Downstream still gets some fish). The ecological threshold is crossed by any H. So the unique strategic tension is the "upstream over-extraction dilemma".
