The payoffs should reflect that defecting gives higher crop but harms fish, and the fish harm affects the defector less if he is downstream. So we can set up the matrix with U and D, where D has first access to fish.

Let's try to create a matrix where (C,C) is better for both than (D,D), but each has a temptation to defect. For U, defecting gives more crop and he doesn't lose much fish because he is last anyway. For D, defecting gives more crop, but he loses fish if the stock collapses. However, because D is first, he might still get fish even in moderate decline. So his temptation is high. But if both defect, fish collapse, and even D's fish income drops significantly. So D might prefer (C,C) over (D,D) if the fish collapse is severe enough.

Let's set parameters:
Crop net per field: 2. (So C=4 fields -> 8; D=8 fields -> 16)
Fish income:
- If total fields <=8 (both C): fish healthy, U fish=10, D fish=10.
- If total fields =12 (one C one D): fish moderate, U fish=2, D fish=10 (D still gets target).
- If total fields =16 (both D): fish collapsed, U fish=0, D fish=4 (even D suffers).

Now payoffs:
C,C: U=8+10=18, D=8+10=18.
C,D: U=8+2=10, D=16+10=26.
D,C: U=16+2=18, D=8+10=18.
D,D: U=16+0=16, D=16+4=20.

Check incentives:
U: if D plays C, U prefers D (18 vs 18? Actually, D,C gives U 18, C,C gives 18; indifferent). If D plays D, U prefers D (16 > 10). So U weakly prefers D.
D: if U plays C, D prefers D (26 > 18). If U plays D, D prefers D (20 > 18). So D strictly prefers D.
Nash equilibrium: (D,D) with payoffs (16,20). Compare to (C,C): (18,18). So U prefers (C,C) (18>16), D prefers (D,D) (20>18). So this is not a PD; it's a game where D's dominant strategy leads to an outcome that U dislikes, but U's best response is also D. So it's a "prisoner's dilemma" for U but not for D. That reflects the asymmetry: D's advantage in fishing makes him less concerned about fish collapse, so he defects, forcing U to defect as well. This is a valid strategic tension.

We could also design a matrix where the threshold is sharper: if total fields > 8, fish stock collapses completely, and both get zero fish. Then:
C,C: total=8, fish healthy, U=18, D=18.
C,D: total=12, fish collapsed, U=8+0=8, D=16+0=16.
D,C: total=12, fish collapsed, U=16+0=16, D=8+0=8.
D,D: total=16, fish collapsed, U=16, D=16.
Now incentives: U: if D plays C, U prefers D (16>8); if D plays D, U prefers D (16>8). So U dominant D. D: if U plays C, D prefers D (16>8); if U plays D, D prefers D (16>8). So both dominant D. Nash (D,D) with (16,16). Compare to (C,C): (18,18). So both prefer (C,C). This is a classic PD! But it lacks the asymmetry in fish access because we assumed collapse gives zero to both.
