So if threshold met, D gets high fish, U gets low fish. If threshold not met, both get 0. So D has more to gain from the threshold being met. That changes the game.

Let's design a matrix for the **Fish Conservation Game** as a distinct action situation. Participants: U and D. Actions: C (conserve, limit irrigation) and NC (not conserve, maximize irrigation). The threshold: if both C, threshold met; if one C and one NC, threshold not met (because total water use is too high); if both NC, threshold not met. So fish benefit only if both C. Agricultural yield: C gives low yield, NC gives high yield. But if U chooses NC, he gets high yield regardless; if U chooses C, he gets low yield. D's yield: if U chooses C, D can get high yield if he chooses NC, but if U chooses NC, D's yield might be reduced because water is taken by U. So we need to specify the water flow.

Let's set the numbers again with the threshold condition: total water use must be ≤ 80 for threshold to be met. Inflow = 100. C uses 30, NC uses 60. So if both C: total 60 ≤ 80, threshold met. If one C and one NC: total 90 > 80, threshold not met. If both NC: total 120 > 80, threshold not met.

Now, agricultural yields: U gets first dibs. So U's yield: if C, he takes 30 -> yield 30; if NC, he takes 60 -> yield 60. D's yield: if U C, water left 70. If D C, he takes 30 -> yield 30; if D NC, he takes 60 -> yield 60. If U NC, water left 40. If D C, he takes 30 -> yield 30 (since 40 available, he gets his 30); if D NC, he takes 60 but only 40 available -> yield 40.

Fish benefit: if threshold met (both C), fish benefit = F_U for U, F_D for D. Since D is first, let F_D > F_U. If threshold not met, fish = 0.

So payoffs = agri yield + fish.
Let F_U = 20, F_D = 40. (Or any numbers)
Then:
- (C, C): U: 30+20=50, D: 30+40=70.
- (C, NC): U: 30+0=30, D: 60+0=60. (Wait, D agri: U C, so water left 70, D NC takes 60 -> yield 60. So D=60, U=30)
- (NC, C): U: 60+0=60, D: 30+0=30. (U NC takes 60, left 40; D C takes 30, gets 30. So U=60, D=30)
- (NC, NC): U: 60+0=60, D: 40+0=40. (U NC takes 60, left 40; D NC takes 40, yield 40. So U=60, D=40)

Matrix:
U\D   C      NC
