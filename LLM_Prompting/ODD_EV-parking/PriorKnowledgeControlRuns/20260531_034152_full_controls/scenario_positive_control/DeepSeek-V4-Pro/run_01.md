### Action Situation 1: Queue-order and prompt move-out compliance between EV users
**Tension:** EV users must decide whether to respect the queue order and move out promptly, or exploit by jumping the queue or overstaying, creating a social dilemma.
**Matrix:**
| EV User A / EV User B | Follow | Exploit |
|---|---|---|
| Follow | (8, 8) | (3, 10) |
| Exploit | (10, 3) | (4, 4) |
**Justification:** This is an action situation because two EV users interact under a posted queue rule, each choosing between compliance and exploitation. The payoffs create a prisoner’s dilemma where individual incentives to exploit can lead to mutually worse outcomes.
