### Action Situation 1: Queue Access and Fairness
#### Title: Queue Order Compliance
#### Tension: Residents vs. Non-Residents on Queue Priority
#### Matrix:
|  | Resident Complies | Resident Bypasses |
| --- | --- | --- |
| **Non-Resident Complies** | R:3, NR:3 | R:5, NR:1 |
| **Non-Resident Bypasses** | R:1, NR:5 | R:2, NR:2 |
#### Justification: This matrix reflects the strategic tension between residents and non-residents regarding queue priority. When both comply, they receive a moderate payoff (3). If a resident bypasses, they gain (5) but the non-resident loses (1). If both bypass, payoffs are low (2), indicating conflict.
