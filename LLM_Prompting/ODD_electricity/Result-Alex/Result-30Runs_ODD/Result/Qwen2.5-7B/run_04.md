### Title: Action Situations in Electricity Governance Model of Andhra Pradesh

#### 1. Farmer-Farmer Coordination (AS1)
**Tension:** Mutual Cooperation vs. Free-Rider Dilemma
**Matrix:**
```
                Farmer 2
                Invest     No Invest
Farmer 1 Invest   (3, 3)     (0, 4)
              No Invest (4, 0)     (2, 2)
```
**Justification:** Two farmers decide whether to invest in voltage-stabilizing equipment. Mutual investment provides a shared benefit, but unilateral investment does not yield any private benefit. This creates a coordination problem where mutual cooperation is Pareto-dominant but risky.

#### 2. Sequential Social Learning (AS2)
**Tension:** Diffusion of Capacitor Adoption
**Sequential Representation (Game Tree):**
```
                Farmer 2
                   / \
                  /   \
             (3,3)  (2,2)
            /   \
           /     \
       (2,2)  (1,1)
```
**Justification:** Each farmer observes a peer’s outcome and imitates only if it ranks higher. This sequential structure allows for the diffusion of capacitor adoption only after a successful coordinated trial has been observed.

#### 3. Asymmetric Transformer Capacity Authorization (AS3)
**Tension:** Free-Rider Dilemma
**Matrix:**
```
                Farmer 2
                Authorize  No Authorize
Farmer 1 Authorize   (3, 3)     (0, 4)
              No Authorize (4, 0)     (2, 2)
```
**Justification:** One farmer’s authorization or investment benefits both by raising voltage quality, but the costs fall solely on the authorizer. This generates a free-rider incentive, making mutual cooperation Pareto-dominant but risky.

#### 4. Mutual-Exchange Coordination (AS4)
**Tension:** Reciprocal Benefit vs. Opportunism
**Matrix:**
```
                Sub-station Staff
                Exchange   No Exchange
Farmer         Exchange   (3, 3)     (0, 2)
              No Exchange (2, 0)     (2, 2)
```
**Justification:** Reciprocal benefit arises only when both a farmer and sub-station staff engage in informal exchange. If either party abstains, the offerer bears a loss while the other reverts to baseline.

#### 5. Asymmetric Authorization and Investment (AS5)
**Tension:** Legal Cooperation vs. Opportunism
**Matrix:**
```
                Sub-station Staff
                Formal Request   Informal Request
Farmer         Formal Request   (3, 3)     (0, 1)
              Informal Request (1, 0)     (2, 2)
```
**Justification:** Mutual formal cooperation is collectively optimal, but if the farmer makes a formal request and the staff withhold, the farmer incurs a loss while the staff save effort or cost. If the farmer makes an informal request and staff invest, the farmer gains more while the staff bear the cost without the formal fee.

#### 6. Groundwater Extraction Prisoner’s Dilemma (AS6)
**Tension:** Mutual Restraint vs. Unilateral Over-Extraction
**Matrix:**
```
                Farmer 2
                Restrain   Over-Extract
Farmer 1 Restrain   (3, 3)     (0, 4)
              Over-Extract (4, 0)     (1, 1)
```
**Justification:** Mutual restraint sustains yields, but unilateral over-extraction offers short-term gain and accelerates depletion.

#### 8. Farmer-Sub-station Staff Coordination (AS8)
**Tension:** Informal vs. Formal Exchange
**Matrix:**
```
                Sub-station Staff
                Formal Exchange   Informal Exchange
Farmer         Formal Exchange   (3, 3)     (0, 1)
              Informal Exchange (1, 0)     (2, 2)
```
**Justification:** Reciprocal benefit arises only when both engage in informal exchange, reflecting a preference for informal over formal interactions.

#### 9. Social Learning (AS9)

#### 10. Bounded Rationality (AS10)
**Tension:** Rational vs. Irrational Behavior
**Matrix:**
```
                Farmer 2

Farmer 1 Invest   (3, 3)     (0, 4)
```
