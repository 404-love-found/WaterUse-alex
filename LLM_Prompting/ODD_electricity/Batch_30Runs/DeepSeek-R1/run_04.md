### Capacitor Adoption Assurance Game  
**Tension:** Farmers must coordinate capacitor adoption for mutual benefit; unilateral adoption yields no private gain due to interdependence.  
**Matrix:**  
```
            Farmer B: Adopt   Farmer B: Not Adopt  
Farmer A: Adopt   (3, 3)          (1, 2)  
Farmer A: Not     (2, 1)          (2, 2)  
```  
**Justification:** Mutual adoption improves voltage stability collectively, but unilateral adoption incurs cost without benefit (AS1). Ordinal ranks reflect Pareto-dominant mutual cooperation versus risky unilateral action.

### Transformer Capacity Authorization Dilemma  
**Tension:** Authorization costs burden one farmer, but benefits (improved voltage) are shared non-excludably, creating free-riding incentives.  
**Matrix:**  
```
            Farmer B: Authorize   Farmer B: Not Authorize  
Farmer A: Authorize   (2, 2)          (-1, 3)  
Farmer A: Not         (3, -1)         (1, 1)  
```  
**Justification:** Non-contributors benefit more than contributors when only one invests. Neither investing maintains low baseline reliability (AS3). Asymmetric payoffs reflect free-rider dynamics.

### Mutual-Exchange Coordination Game  
**Tension:** Informal reciprocity requires matched cooperation; mismatched actions penalize the cooperating party.  
**Matrix:**  
```
            Staff: Cooperate   Staff: Defect  
Farmer: Cooperate   (3, 3)          (0, 2)  
Farmer: Defect      (2, 0)          (2, 2)  
```  
**Justification:** Mutual engagement yields reciprocal gain (e.g., collusion). Unilateral cooperation imposes loss on the offerer while the defector retains baseline payoffs (AS4). Stag Hunt structure captures coordination fragility.

### Authorization-Investment Asymmetric Game  
**Tension:** Formal cooperation optimizes collective outcomes, but asymmetric incentives favor opportunism (farmer: informal access; staff: effort withholding).  
**Matrix:**  
```
            Staff: Invest   Staff: Withhold  
Farmer: Formal     (3, 2)          (0, 3)  
Farmer: Informal   (4, 1)          (2, 2)  
```  
**Justification:** Formal-investment alignment is collectively optimal but privately costly for staff. Farmer gains more from informal requests if staff invest, while staff gain from withholding regardless (AS5).

### Groundwater Extraction Prisoner's Dilemma  
**Tension:** Restraint sustains groundwater, but unilateral over-extraction yields higher short-term gains, accelerating depletion.  
**Matrix:**  
```
            Farmer B: Restrain   Farmer B: Extract  
Farmer A: Restrain   (3, 3)          (0, 4)  
Farmer A: Extract    (4, 0)          (1, 1)  
```  
**Justification:** Mutual extraction depletes aquifers, raising pumping costs. Unilateral restraint sacrifices individual gain (AS6). Classic PD structure with T > R > P > S.
