### Capacitor-Adoption Assurance Game (Two Farmers)
**Tension**: Coordination problem where mutual investment in capacitors yields shared voltage improvement, but unilateral investment provides no private benefit and incurs cost, creating a risk of coordination failure despite Pareto-dominant mutual cooperation.  
**Matrix**:  
```
          Farmer2: Adopt   Farmer2: Not  
Farmer1: Adopt   (3, 3)        (1, 2)  
Farmer1: Not     (2, 1)        (2, 2)  
```  
**Justification**: Based on AS1 description: Mutual adoption improves voltage stability (Pareto-dominant), but unilateral adoption fails due to no private benefit (low payoff for adopter). Non-adoption yields status quo (intermediate payoff). Ordinal ranks reflect asymmetric costs/benefits per empirical incentives.

### Transformer-Capacity Authorization Dilemma (Two Farmers)  
**Tension**: Asymmetric free-rider problem where one farmer’s investment in transformer capacity benefits both, but non-investors gain more, discouraging voluntary contributions despite collective benefits.  
**Matrix**:  
```
          Farmer2: Invest   Farmer2: Not  
Farmer1: Invest   (2, 2)        (2, 3)  
Farmer1: Not      (3, 2)        (1, 1)  
```  
**Justification**: Derived from AS3: Investor bears full cost while non-investor free-rides (higher payoff). Neither investing maintains low baseline. Ordinal payoffs capture uneven costs and free-riding incentives observed in Andhra Pradesh.

### Mutual-Exchange Coordination (Farmer and Staff)  
**Tension**: Reciprocal gains only if both engage in informal exchange; mismatched actions penalize the cooperator (e.g., farmer offering bribe or staff tolerating unauthorized connections), creating a coordination equilibrium.  
**Matrix**:  
```
          Staff: Engage    Staff: Abstain  
Farmer: Engage   (4, 4)        (1, 3)  
Farmer: Abstain  (3, 1)        (2, 2)  
```  
**Justification**: AS4 description: Mutual exchange maximizes informal benefits (e.g., collusion). Unilateral cooperation incurs loss (e.g., farmer penalized if staff enforces). Abstention yields baseline. Ranks reflect field-observed reciprocity norms.

### Authorization-Investment Asymmetric Coordination (Farmer and Staff)  
**Tension**: Conflicting incentives between formal compliance (collectively optimal) and opportunism; farmer/staff gains diverge under mixed strategies (e.g., informal requests vs. withheld investment).  
**Matrix**:  
```
          Staff: Invest    Staff: Withhold  
Farmer: Formal   (3, 3)        (1, 4)  
Farmer: Informal (4, 1)        (2, 2)  
```  
**Justification**: AS5 logic: Formal cooperation (Farmer:Formal, Staff:Invest) is optimal but modest for staff due to effort. Withholding investment saves staff effort (high payoff) but penalizes farmer. Informal requests exploit opportunities asymmetrically.

### Groundwater-Extraction Prisoner’s Dilemma (Two Farmers)  
**Tension**: Short-term individual gain from over-extraction vs. collective long-term sustainability; mutual extraction accelerates depletion despite mutual restraint being optimal.  
**Matrix**:  
```
          Farmer2: Restrain   Farmer2: Extract  
Farmer1: Restrain   (3, 3)        (1, 4)  
Farmer1: Extract    (4, 1)        (2, 2)  
```  
**Justification**: AS6 structure: Unilateral extraction dominates (high payoff) but mutual extraction depletes groundwater (worse than restraint). Ranks reflect short-term incentives and aquifer depletion dynamics per empirical data.
