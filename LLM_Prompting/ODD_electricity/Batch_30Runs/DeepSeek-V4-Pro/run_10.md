**AS1: Capacitor Adoption Assurance Game (Simultaneous)**
- Title: Simultaneous Capacitor Adoption Assurance Game
- Tension: Mutual adoption yields shared improvement, but unilateral adoption is costly and ineffective; thus, adoption is risky without coordination.
- Matrix (Farmer 1, Farmer 2):
        Adopt   Not
Adopt   (3,3)   (1,2)
Not     (2,1)   (2,2)
But wait, in stag hunt, (Not, Adopt) is (2,1)? Actually, if Farmer 1 does not adopt and Farmer 2 adopts, Farmer 1 gets 2 (baseline), Farmer 2 gets 1 (cost). So the matrix is:
         Adopt    Not
Adopt   (3,3)   (1,2)
Not     (2,1)   (2,2)
But is (Not, Adopt) = (2,1) or (2,2)? In stag hunt, the non-adopter gets the hare (2), the adopter gets nothing (1). So (Not, Adopt) = (2,1). So the matrix is exactly the stag hunt. But the ODD+D says "mutual cooperation Pareto-dominant but risky". In stag hunt, (Adopt, Adopt) is (3,3) and (Not, Not) is (2,2). So (3,3) > (2,2). The risky part is that if you adopt and the other doesn't, you get 1. So the matrix is correct. We'll use 3 and 2 as the cooperative and baseline payoffs. Some might use 4 and 2, but we can use 3 and 2. To make it ordinal, we can use 4=best, 1=worst. Let's set: (Adopt, Adopt) = (4,4); (Adopt, Not) = (1,2); (Not, Adopt) = (2,1); (Not, Not) = (2,2). But then (Not, Not) is 2, and (Adopt, Not) the non-adopter gets 2. So the non-adopter gets the same payoff whether the other adopts or not. That's consistent with "unilateral investment yields no added private benefit" meaning the non-adopter doesn't get a benefit either. So the non-adopter's payoff is always 2. The adopter's payoff is 4 if both adopt, 1 if only he adopts. So the matrix:
        Adopt   Not
Adopt: (4,4)  (1,2)
Not:   (2,1)  (2,2)
This is a stag hunt. We'll use this.
