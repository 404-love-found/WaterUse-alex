| --- | --- | --- |
| **Farmer 2 Authorizes** | 2, 2 | 4, 1 |
| **Farmer 2 Does Not Authorize** | 1, 4 | 0, 0 |
#### Justification: This situation represents the asymmetric transformer-capacity authorization dilemma. One farmer's authorization benefits both but costs only the authorizer, generating a free-rider incentive. If only one invests, the contributor bears the cost while the non-investor benefits more.
