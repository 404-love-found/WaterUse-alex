### Title: Action Situations in the Irrigation Electricity Governance Model

#### 1. Farmer-Farmer Coordination: Capacitor Adoption Assurance Game (AS1)
**Tension:** Mutual Cooperation vs. Free-Rider Dilemma

**Matrix:**
```
          Farmer B
          |       |
          | 1,1   | 0,0
          |--------|
          | 0,0   | 0,0
```
**Justification:** Two neighboring farmers decide whether to invest in voltage-stabilizing capacitors. Mutual investment yields a shared improvement, but unilateral investment offers no private benefit, creating a coordination problem where mutual cooperation is Pareto-dominant but risky.

#### 2. Farmer-Utility Staff Mutual-Exchange Coordination Game (AS4)
**Tension:** Informal Exchange vs. Formal Compliance

**Matrix:**
```
          Staff
          |       |
          | 1,1   | 0,1
          |--------|
          | 1,0   | 0,0
```
**Justification:** A farmer and sub-station staff engage in a game where mutual benefit arises only when both engage in informal exchanges. If either party abstains, the offerer bears a loss while the abstainer reverts to baseline, and if both abstain, no extra benefit occurs.

#### 3. Farmer-Farmer Authorization and Investment Asymmetric Coordination Game (AS5)
**Tension:** Formal Cooperation vs. Opportunism

**Tree:**
```
                         Farmer
                            |
                       Formal Request
                         /       \
                      Staff Invest Staff Withhold
                         |           |
                Farmer Gains, Staff Slightly Loses Farmer Loses, Staff Slightly Saves
```
**Justification:** A farmer can request formal or informal authorization for capacitor installation, while the staff can invest or withhold capacity. Mutual formal cooperation is collectively optimal, but if the farmer requests formally and staff withholds, the farmer incurs a loss, and if the farmer requests informally and staff invests, the farmer gains while the staff bears the cost without the formal fee.

#### 4. Farmer-Farmer Groundwater Extraction Prisoner's Dilemma (AS6)
**Tension:** Mutual Restraint vs. Unilateral Over-Extraction

**Matrix:**
```
          Farmer B
          |       |
          | 1,1   | 0,2
          |--------|
          | 2,0   | 0,0
```
**Justification:** Two farmers drawing from the same aquifer face a dilemma where mutual restraint sustains yields, but unilateral over-extraction offers short-term gain but accelerates depletion.

#### 5. Farmer-Farmer Sequential Social-Learning Process (AS2)
**Tension:** Adoption of Capacitor Measures vs. Free-Rider Dilemma

**Tree:**
```
                         Farmer A
                            |
                       Observe Outcome B
                         /       \
                    Imitate if B > A Stay Unchanged
```
**Justification:** Farmer A observes the outcome of Farmer B's capacitor adoption and imitates only if B’s outcome ranks higher. Diffusion occurs only after a successful coordinated trial has been observed.

#### 6. Farmer-Staff Asymmetric Transformer-Capacity Authorization Dilemma (AS3)
**Tension:** Farmer's Investment vs. Staff's Capacity Provision

**Matrix:**
```
          Staff
          |       |
          | 1,1   | 0,0
          |--------|
          | 0,0   | 0,0
```
**Justification:** One farmer’s authorization or investment benefits both by raising voltage quality, but costs fall solely on the authorizer, generating a free-rider incentive and uneven payoffs.
