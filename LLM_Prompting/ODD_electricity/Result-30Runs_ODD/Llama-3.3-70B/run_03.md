#### Title: Mutual Investment in Voltage-Stabilising Equipment
#### Tension: Coordination Problem
#### Matrix/Sequential Representation:
|  | Farmer 2 Invests | Farmer 2 Does Not Invest |
| --- | --- | --- |

#### Tension: Social Learning Dilemma
#### Matrix/Sequential Representation:
- Farmer 1 adopts capacitor
  - Farmer 2 observes outcome and imitates if outcome ranks higher
  - Farmer 2 does not imitate if outcome does not rank higher

#### Justification: This action situation represents the sequential social-learning process where each farmer observes a peer’s outcome and imitates only if that outcome ranks higher.
