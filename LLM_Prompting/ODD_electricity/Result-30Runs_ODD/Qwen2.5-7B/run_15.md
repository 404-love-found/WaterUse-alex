### Title: Action Situations in Irrigation Electricity Governance Model

#### 1. Farmer-Farmer Coordination: Capacitor Adoption Assurance Game (AS1)

**Tension:**
Mutual cooperation is Pareto-dominant, but risky due to the potential for unilateral defection.

**Matrix Representation:**
```
                Farmer B
                Invest   No Invest
Farmer A
Invest         (1, 1)    (0, 2)
No Invest      (2, 0)    (0, 0)
```

**Justification:**
This game captures the tension where mutual investment in capacitors provides a shared benefit, but unilateral investment yields no additional private benefit, leading to a coordination problem.

#### 2. Sequential Social-Learning Process in Capacitor Adoption (AS2)

**Tension:**
Diffusion of capacitor adoption occurs only after a successful coordinated trial has been observed.

**Sequential Representation:**
```
Farmer A
\       / 
  Invest     No Invest
\       /
Farmer B
```

**Justification:**
This sequential game reflects that diffusion happens only after a successful coordinated trial, illustrating the social learning and bounded rationality.

#### 3. Asymmetric Transformer-Capacity Authorization Dilemma (AS3)

**Tension:**
One farmer’s authorization benefits both, but costs fall solely on the authorizer, creating a free-rider incentive.

**Matrix Representation:**
```
                Farmer B
                Authorize   No Authorize
Farmer A
Authorize       (1, 1)       (2, 0)
No Authorize    (0, 2)       (0, 0)
```

**Justification:**
This game highlights the asymmetric costs and benefits of transformer capacity authorization, where one farmer bears the cost while both benefit.

#### 4. Mutual-Exchange Coordination Game (AS4)

**Tension:**
Reciprocal benefit arises only when both parties engage in informal exchange.

**Matrix Representation:**
```
                Sub-station
                Exchange   No Exchange
Farmer
Exchange       (1, 1)       (0, 0)
No Exchange    (0, 0)       (1, 1)
```

**Justification:**
This game captures the mutual exchange coordination where both parties must engage for mutual benefit.

#### 5. Authorization-and-Investment Asymmetric Coordination Game (AS5)

**Tension:**
Mutual formal cooperation is collectively optimal, but staff have incentives to withhold capacity under informal requests.

**Matrix Representation:**
```
                Sub-station
                Formal Request   Informal Request
Farmer
Formal Request  (1, 1)       (0, 1)
Informal Request (1, 0)       (2, 0)
```

**Justification:**
This game reflects the asymmetric incentives between legality and opportunism, where staff gain modestly even under formal cooperation.

#### 6. Groundwater Extraction Prisoner’s Dilemma (AS6)

**Tension:**
Mutual restraint sustains yields, but unilateral over-extraction offers short-term gain and accelerates depletion.

**Matrix Representation:**
```
                Farmer B
                Restrain    Over-extract
Farmer A
Restrain        (1, 1)       (0, 2)
Over-extract    (2, 0)       (0, 0)
```

**Justification:**
This game captures the tension where mutual restraint provides sustainable yields, but unilateral over-extraction offers short-term benefits at the cost of long-term depletion.
