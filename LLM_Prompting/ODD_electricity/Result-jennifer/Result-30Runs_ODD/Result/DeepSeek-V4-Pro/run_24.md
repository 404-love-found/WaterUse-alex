3. Authorization: Asymmetric trust dilemma (staff dominant strategy to deny, farmer best response not apply). This is like a one-sided PD where staff's defection is dominant, leading to suboptimal Nash. It's not a standard symmetric game.
4. Regularization: Asymmetric conflict (farmer dominant strategy to reject, staff best response not offer). Nash is (Reject, Not offer), which is suboptimal for staff. This is a deadlock game where one player's dominant strategy leads to a bad outcome for the other, but not a social dilemma in the sense of both being worse off than some other outcome. Actually farmer gets 2 in both (Reject, Not offer) and (Accept, Offer) – we set (Accept, Offer) farmer 2, (Reject, Not offer) farmer 2. So farmer is indifferent, but staff is worse. So it's a coordination failure with asymmetric payoffs. Could be seen as a "Battle of the Sexes" where both prefer different equilibria, but here farmer's best outcome is (Reject, Offer) 3, which is not an equilibrium. So it's a mixed-motive game. Distinct.
5. Enforcement: Inspection game with no pure Nash, cyclic. Distinct.

All five strategic games are distinct in their incentive structures: stag hunt, PD, asymmetric trust (staff dominant defect), asymmetric conflict (farmer dominant defect), cyclic inspection. No two are identical. So diversity satisfied.
