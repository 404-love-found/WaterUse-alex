### Title: Distinct Action Situations in the Irrigation Electricity Governance Model

#### 1. **Farmer-Farmer Capacitor Adoption Assurance Game (AS1)**
   - **Tension:** Mutual Coordination vs. Free-Rider Dilemma
   - **Matrix:**
     ```
     |           | Farmer 2: Adopt | Farmer 2: Not Adopt |
     |-----------|----------------|--------------------|
     | Farmer 1: Adopt | 2, 2            | 1, 3               |
     | Farmer 1: Not Adopt | 3, 1            | 1, 1               |
     ```
   - **Justification:** Both farmers benefit from mutual capacitor adoption (mutual cooperation Pareto-dominant), but unilateral adoption offers no private benefit, creating a free-rider incentive.

#### 2. **Farmer-Staff Mutual-Exchange Coordination Game (AS4)**
   - **Tension:** Formal vs. Informal Exchange
   - **Matrix:**
     ```
     |           | Staff: Invest | Staff: Withhold |
     |-----------|--------------|----------------|
     | Farmer: Invest | 2, 2         | 1, 0            |
     | Farmer: Withhold | 0, 1         | 1, 1            |
     ```
   - **Justification:** Mutual formal cooperation yields reciprocal benefit, but if either party abstains, the offerer incurs a loss, and the other reverts to baseline. Informal exchanges provide modest gains for staff.

#### 3. **Farmer-Farmer Groundwater Extraction Prisoner's Dilemma (AS6)**
   - **Tension:** Mutual Restraint vs. Unilateral Extraction
   - **Matrix:**
     ```
     |           | Farmer 2: Restrain | Farmer 2: Extract |
     |-----------|--------------------|-------------------|
     | Farmer 1: Restrain | 2, 2               | 1, 3              |
     | Farmer 1: Extract   | 3, 1               | 1, 1              |
     ```
   - **Justification:** Mutual restraint sustains groundwater yields, but unilateral extraction offers short-term gain and accelerates depletion, creating a classic prisoner's dilemma.

#### 4. **Farmer-Farmer Transformer Capacity Authorization Dilemma (AS3)**
   - **Tension:** Mutual Investment vs. Free-Rider Incentive
   - **Matrix:**
     ```
     |           | Farmer 2: Invest | Farmer 2: Not Invest |
     |-----------|-----------------|---------------------|
     | Farmer 1: Invest | 2, 2            | 1, 3                |
     | Farmer 1: Not Invest | 3, 1            | 1, 1                |
     ```
   - **Justification:** Mutual investment in transformer capacity improves voltage quality, but costs fall solely on the authorizer, creating a free-rider incentive.

#### 5. **Farmer-Staff Authorization and Investment Asymmetric Coordination Game (AS5)**
   - **Tension:** Formal vs. Informal Request and Investment
   - **Matrix:**
     ```
     |           | Staff: Invest | Staff: Withhold |
     |-----------|--------------|----------------|
     | Farmer: Formal Request | 2, 2         | 1, 0            |
     | Farmer: Informal Request | 0, 1         | 1, 1            |
     ```
   - **Justification:** Formal cooperation is collectively optimal, but if the farmer makes a formal request and staff withhold, the farmer incurs a loss. Informal requests and investments yield modest gains for staff.

#### 6. **Farmer-Farmer Sequential Social-Learning Process (AS2)**
   - **Tension:** Diffusion of Capacitor Adoption
   - **Matrix:**
     - This is a sequential game where each farmer imitates a peer only if the peer's outcome ranks higher. No direct payoff matrix exists, but the process can be described as follows:
     - Farmer 1 observes Farmer 2's outcome and imitates if the outcome is better.
     - If both farmers adopt, they both benefit. If one adopts and the other does not, the adopter benefits more. If neither adopts, both remain at a low baseline.
