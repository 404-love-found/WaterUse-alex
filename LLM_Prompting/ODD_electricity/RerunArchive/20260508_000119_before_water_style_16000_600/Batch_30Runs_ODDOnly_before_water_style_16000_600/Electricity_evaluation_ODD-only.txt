  FN = How many correct ASs the LLM missed
  FP = How many LLM-generated ASs were not in the correct set of ASs
  Precision = TP / (TP + FP)
  Recall    = TP / (TP + FN)

Scoring rules:
  Each generated AS is reviewed against only the six confirmed correct ASs.
  The first generated AS matching a correct AS counts as TP.
  Additional generated ASs matching the same correct AS count as FP-duplicate.
  Generated ASs matching none of the correct ASs count as FP-wrong.
  Missing correct ASs count as FN, so TP + FN = 6 for every run.
  Generated AS titles are included with body/matrix/sequential text when judging correctness.
