### Action Situation: Capacitor Adoption Assurance (AS1)

**Title**: Capacitor Adoption Assurance

**Tension**: Coordination Dilemma

**Matrix/Sequential Representation**:
```
                Farmer B
              _________
             |         |
             |  2,2   |  0,1
             |         |
             |_________|
             |         |
             |  1,0   |  1,1
             |         |
             |_________|
```

**Justification**:
This action situation involves two neighboring farmers deciding whether to invest in voltage-stabilizing capacitors. Mutual investment yields shared improvement in voltage quality, but unilateral investment yields no added private benefit. The payoff matrix reflects the coordination problem where mutual cooperation is Pareto-dominant but risky. If both farmers invest, they both benefit (2,2). If only one invests, the investor incurs a cost (1,0) while the non-investor benefits (1,0). If neither invests, both remain at a low but non-zero baseline (1,1).

### Action Situation: Sequential Social-Learning Process (AS2)

**Title**: Sequential Social-Learning Process

**Tension**: Path-Dependent Diffusion

**Sequential Representation**:
```
1. Farmer A decides (C: Capacitor, N: No Capacitor)
2. Farmer B observes A's decision and decides (C: Capacitor, N: No Capacitor)
```

**Justification**:
This action situation involves a sequential process where Farmer A decides whether to invest in a capacitor. Farmer B then observes A’s decision and decides whether to invest based on whether A’s decision ranks higher. The process is path-dependent, and early failed or isolated adoption can discourage later uptake, while visibly successful coordinated adoption can spread through the social network.

```
                Farmer A
              _________

             |         |
             |_________|
             |         |
             |  3,1   |  1,1
             |         |
             |_________|
```

**Justification**:
This action situation involves a groundwater extraction prisoner's dilemma where mutual restraint sustains yields but unilateral over-extraction offers short-term gain and accelerates depletion.
