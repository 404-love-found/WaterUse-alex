### Title: Sequential Social-Learning Process in Capacitor Adoption (AS2)

### Tension: Path Dependence in Diffusion
**Justification:** Diffusion of capacitor adoption is path-dependent, and early failures can discourage later uptake.

#### Sequential Game Tree

1. **Farmer 1 decides whether to adopt a capacitor.**
   - If **No Capacitor**: 
     - **Farmer 2 decides whether to adopt a capacitor based on observation.**
       - If **No Capacitor**: (0, 0)
       - If **Capacitor**: (1, -1)
   - If **Capacitor**: 
     - **Farmer 2 decides whether to adopt a capacitor based on observation.**
       - If **No Capacitor**: (-1, 1)
       - If **Capacitor**: (2, 2)
