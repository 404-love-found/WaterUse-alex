### Title: Capacitor Adoption Assurance Game (AS1)

### Tension: Mutual Cooperation vs. Free-Rider Incentive

### Matrix Representation:
```
                  Farmer 2
                  Install Capacitor  |  Do Not Install Capacitor
                  ----------------------------------------------
Farmer 1 Install Capacitor | (4, 4)            | (2, 6)
                  ----------------------------------------------
Farmer 1 Do Not Install Capacitor | (6, 2)            | (3, 3)
```

### Justification:
The Capacitor Adoption Assurance Game (AS1) involves two neighboring farmers deciding whether to install voltage-stabilizing capacitors. Mutual installation yields shared improvement in voltage quality, which is beneficial to both farmers. However, unilateral installation offers no private benefit, creating a free-rider incentive. The payoff matrix reflects the mutual benefit of coordinated action and the disadvantage of free-riding.

### Title: Asymmetric Transformer-Capacity Authorization Dilemma (AS3)

### Tension: Informal vs. Formal Access

### Sequential Representation (Game Tree):

```
        Farmer 1
         /   \
    Formal Access  |  Informal Access
         \   /
        Sub-station Staff
         /   \
  Formal Authorization  |  Informal Tolerance
         \   /
        Farmer 2
         /   \
  Formal Access  |  Informal Access
```

### Payoffs:
- Formal Authorization: (5, 5)
- Informal Tolerance: (4, 4)
- Informal Access: (3, 3)
- No Access: (2, 2)

### Justification:
The Asymmetric Transformer-Capacity Authorization Dilemma (AS3) involves a farmer’s decision to request formal access or informal access to transformer capacity, and the sub-station personnel’s decision to authorize or tolerate informal access. Mutual formal authorization is collectively optimal, but unilateral informal access offers short-term benefits for one party while imposing costs on the other. The sequential game tree captures the dynamic decision-making process and the resulting payoffs for both parties.

### Title: Groundwater Extraction Prisoner's Dilemma (AS6)

### Tension: Cooperative vs. Competitive Extraction

### Matrix Representation:
```
                  Farmer 2
                  Extract Groundwater  |  Do Not Extract Groundwater
                  ----------------------------------------------
Farmer 1 Extract Groundwater | (2, 2)            | (5, 1)
                  ----------------------------------------------
Farmer 1 Do Not Extract Groundwater | (1, 5)            | (4, 4)
```

### Justification:
The Groundwater Extraction Prisoner's Dilemma (AS6) involves two farmers drawing from the same aquifer. Mutual restraint sustains yields, but unilateral over-extraction offers short-term gain and accelerates depletion. The payoff matrix reflects the short-term benefit of competitive extraction versus the long-term benefit of cooperative restraint.
