### Title: Capacitor Adoption Assurance Game (AS1)

### Tension: Mutual Coordination vs. Unilateral Investment

### Matrix/Sequential Representation:
**Normal Form Payoff Matrix:**

|                | Farmer 2: No Capacitor | Farmer 2: Capacitor |
|----------------|------------------------|---------------------|
| **Farmer 1: No Capacitor** | (0, 0)                 | (1, -1)              |
| **Farmer 1: Capacitor**     | (-1, 1)                | (2, 2)               |

### Justification:
This action situation captures the strategic tension between mutual coordination and unilateral investment in capacitors. If both farmers install capacitors, the voltage stability improves, benefiting both (2, 2). If one farmer invests and the other does not, the investing farmer bears the cost (1, -1) or (-1, 1), while the non-investing farmer benefits from the improved voltage (2, 2). If neither invests, there is no improvement (0, 0).

### Title: Sequential Social-Learning Process in Capacitor Adoption (AS2)

### Tension: Sequential Adoption vs. Immediate Cooperation

### Sequential Representation (Game Tree):

```
                  [Farmer 1]
                   / \
                  /   \
               No Cap  Cap
               (0,0) (1,1)
                  / \
                 /   \
            [Farmer 2]
             / \
            /   \
         No Cap  Cap
         (0,0) (1,1)
```

### Justification:
This action situation involves a sequential process where Farmer 1 can either adopt a capacitor or not. If Farmer 1 adopts, Farmer 2 observes and decides whether to adopt based on the outcome. If the first farmer's adoption does not yield a visible improvement (0,0), the second farmer is less likely to adopt (0,0). If the first farmer adopts and it visibly improves voltage (1,1), the second farmer is more likely to adopt (1,1). The decision process is influenced by social learning and the visibility of outcomes.

### Title: Farmer-Sub-Station Personnel Interaction (AS7)

```
                   / \
                  /   \
               No Cap  Cap

                  / \
                 /   \
            [Sub-Station Personnel]
             / \
            /   \
         Formal  Informal
         (0,0) (1,1)
```

### Justification:
This action situation captures the strategic tension between formal compliance and informal exchange. If the farmer requests formal access and the sub-station personnel invest, the farmer benefits (1,1) and the personnel benefit (1,1). If the farmer requests informal access and the personnel enforce, the farmer suffers (0,0) and the personnel benefit (1,1).

### Title: Bounded Rationality and Social Learning (AS10)

### Tension: Perfect Knowledge vs. Heuristic Learning

### Matrix/Sequential Representation:
**Normal Form Payoff Matrix:**

|                | Farmer 2: Correct Attribution | Farmer 2: Incorrect Attribution |
|----------------|--------------------------------|----------------------------------|
| **Farmer 1: Correct Attribution** | (2, 2)                         | (1, 1)                           |
| **Farmer 1: Incorrect Attribution** | (1, 1)                         | (0, 0)                           |

### Justification:
This action situation captures the strategic tension between perfect knowledge and heuristic learning. If both farmers correctly attribute the cause of voltage drops or pump failures, they can make better decisions (2, 2). If one farmer incorrectly attributes the cause and the other correctly attributes, the decision quality is lower (1, 1). If both farmers incorrectly attribute the cause, the decision quality is the lowest (0, 0).
