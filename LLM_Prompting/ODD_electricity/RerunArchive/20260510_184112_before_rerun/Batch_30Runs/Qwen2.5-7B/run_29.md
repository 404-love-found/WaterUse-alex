### Action Situation 1: Capacitor Adoption Assurance (AS1)

#### Tension
**Tension:** Mutual Coordination vs. Unilateral Investment

#### Matrix Representation
|                  | Farmer B Invests | Farmer B Does Not Invest |
|------------------|------------------|-------------------------|
| **Farmer A Invests** | (2, 2)           | (1, 0)                  |
| **Farmer A Does Not Invest** | (0, 1)          | (1, 1)                  |

#### Justification
This action situation models the mutual coordination dilemma where both farmers need to invest in capacitors to improve voltage stability. If both invest, they benefit equally. If only one invests, the other farmer does not benefit, making unilateral investment unattractive.

### Action Situation 2: Sequential Social-Learning Process (AS2)

#### Tension
**Tension:** Diffusion of Successful Adoptions vs. Path Dependence

#### Sequential Representation (Game Tree)
```
Farmer A
       / \
      /   \
  A Invest  A Does Not Invest
     |        |
     |        |
Farmer B
       / \
      /   \
  B Invest  B Does Not Invest
```

#### Justification
This action situation involves a sequential social-learning process where Farmer A observes the outcome of Farmer B’s capacitor investment. If the observed outcome is successful, Farmer A is more likely to invest. However, if an early adoption fails, it can discourage later adoption, creating a path-dependent diffusion process.

### Action Situation 3: Asymmetric Transformer-Capacity Authorization Dilemma (AS3)

#### Tension
**Tension:** Formal Contribution vs. Informal Tolerance

#### Matrix Representation
|                  | Farmer B Authorizes | Farmer B Does Not Authorize |
|------------------|---------------------|----------------------------|
| **Farmer A Authorizes** | (2, 2)              | (1, 0)                     |
| **Farmer A Does Not Authorize** | (0, 1)          | (1, 1)                     |

#### Justification
This action situation models the dilemma where one farmer’s authorization can improve voltage quality for both, but only if the other farmer also contributes. If only one farmer authorizes, the other benefits at no cost, creating a free-rider incentive and uneven payoffs.

### Action Situation 4: Mutual-Exchange Coordination Game (AS4)

#### Tension
**Tension:** Reciprocal Benefit vs. Non-Reciprocal Loss

#### Matrix Representation
|                  | Staff Invests | Staff Does Not Invest |
|------------------|---------------|----------------------|
| **Farmer Invests** | (2, 2)        | (1, 0)               |
| **Farmer Does Not Invest** | (0, 1)      | (1, 1)               |

#### Justification
This game represents the coordination between a farmer and sub-station personnel where mutual investment in informal exchanges benefits both. If one side abstains, the other side incurs a loss, and no mutual benefit is realized.

### Action Situation 5: Authorization and Investment Asymmetric Coordination Game (AS5)

#### Tension
**Tension:** Formal Cooperation vs. Opportunistic Behavior

#### Matrix Representation
|                  | Farmer Formal Request | Farmer Informal Request |
|------------------|-----------------------|------------------------|
| **Staff Invests** | (2, 2)                | (1, 1)                 |
| **Staff Withholds** | (0, 1)               | (0, 0)                 |

#### Justification
This action situation models the asymmetric coordination where formal cooperation is collectively optimal, but unilateral formal requests can lead to losses for the farmer if the staff withholds. Informal requests can be more beneficial for the farmer but may lead to staff losses if not reciprocated.

### Action Situation 6: Groundwater Extraction Prisoner’s Dilemma (AS6)

#### Tension
**Tension:** Mutual Restraint vs. Unilateral Over-Extraction

#### Matrix Representation
|                  | Farmer B Restraint | Farmer B Over-Extracts |
|------------------|--------------------|-----------------------|
| **Farmer A Restraint** | (2, 2)            | (1, 0)                |
| **Farmer A Over-Extracts** | (0, 1)         | (1, 1)                |

#### Justification
This game models the dilemma where mutual restraint sustains groundwater yields, but unilateral over-extraction can offer short-term gains and accelerate depletion. If both farmers restrain, they benefit equally. If one over-extracts, the other benefits in the short run but faces depletion risks.

### Action Situation 7: Farmer-Sub-Station Personnel Interaction (AS7)

#### Tension
**Tension:** Formal Compliance vs. Informal Exchange

#### Sequential Representation (Game Tree)
```
Farmer
       / \
      /   \
  Request Formal  Request Informal
     |        |
     |        |
Sub-Station Personnel
       / \
      /   \
  Invest  Withhold
```

#### Justification
This action situation models the interaction between farmers and sub-station personnel where formal compliance involves authorization fees and effort costs, while informal exchange involves tolerance of unauthorized access. The decision depends on oversight risk, trust networks, and expected personal benefit.
