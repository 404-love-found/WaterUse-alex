1. **Title**: Capacitor Adoption Assurance Game (AS1)  
   **Tension**: Mutual investment in capacitors yields shared benefits, but unilateral investment is ineffective and costly.  
   **Matrix**:  
   ```
            Farmer B: Invest    Not Invest
   Farmer A:  
     Invest     (High, High)    (Low, Medium)  
     Not Invest (Medium, Low)   (Medium, Medium)  
   ```  
   **Justification**: "Mutual investment yields shared improvement, while unilateral investment yields no added private benefit, creating a coordination problem" (III.iv.a).

2. **Title**: Sequential Social Learning in Capacitor Adoption (AS2)  
   **Tension**: Farmers imitate peers only if outcomes are superior, requiring successful coordinated trials for diffusion.  
   **Sequential Representation**:  
   ```
   Farmer A adopts capacitor → Outcome observed by Farmer B  
   └─ If outcome > Farmer B's status → Farmer B adopts  
   └─ Else → Farmer B rejects adoption  
   ```  
   **Justification**: "Diffusion occurs only after a successful coordinated trial has been observed" (III.iv.a).

3. **Title**: Transformer Capacity Authorization Dilemma (AS3)  
   **Tension**: Authorization benefits all but costs only the contributor, creating free-riding incentives.  
   **Matrix**:

   ```
   Farmer A:  
     Authorize    (Medium, Medium)  (Medium, High)  
     Not Authorize (High, Medium)   (Low, Low)  
   ```  
   **Justification**: "One farmer’s authorization benefits both by raising voltage quality, but costs fall solely on the authorizer" (III.iv.a).

4. **Title**: Mutual Exchange Coordination (AS4)
