Based on the ODD+D protocol, here are the distinct action situations reflecting unique strategic tensions in electricity governance:

1. **Title**: Capacitor-Adoption Assurance Game  
**Tension**: Coordination failure risk in mutual capacitor investment despite Pareto-dominant cooperation.  
**Matrix**:  
```
        Farmer B  
        Adopt    Not Adopt  
Farmer A  
  Adopt    3,3        1,2  
  Not Adopt 2,1        2,2  
```  
**Justification**: AS1 in III.iv.a – Mutual capacitor adoption yields shared voltage stability, but unilateral adoption provides no private benefit (ordinal payoffs reflect coordination incentives).

2. **Title**: Sequential Social-Learning Process  
**Tension**: Sequential adoption dependency where diffusion requires observed success from peers.  
**Sequential Representation**:  
```  
Farmer A adopts capacitor → Outcome observed (success/failure) → Farmer B adopts only if outcome > current status  
```  
**Justification**: AS2 in III.iv.a – Farmers imitate peers post-observation; diffusion hinges on successful coordinated trials.

3. **Title**: Transformer-Capacity Authorization Dilemma  
**Tension**: Free-rider problem where non-investors benefit from others' costly upgrades.  
**Matrix**:  
```
        Farmer B  
        Authorize    Not Authorize  
Farmer A  
  Authorize   B-C, B-C      B-C, B  
  Not Authorize  B, B-C        L, L  
```  
*(B > B-C > L)*  
**Justification**: AS3 in III.iv.a – One farmer’s authorization benefits both, but costs are privately borne, creating asymmetric free-riding.

4. **Title**: Mutual-Exchange Coordination  
**Tension**: Reciprocal gains only when both farmer and staff engage informally; unilateral action incurs losses.  
**Matrix**:  
```
        Staff  
        Engage    Abstain  
Farmer  
  Engage    3,3        -1,0  
  Abstain   0,-1        0,0  
```  
**Justification**: AS4 in III.iv.a – Informal exchanges yield mutual benefit iff both participate; defection harms the initiator.

5. **Title**: Authorization-Investment Asymmetric Coordination  
**Tension**: Tension between formal compliance (collective optimum) and informal opportunism with uneven burdens.  
**Matrix**:  
```
        Staff  
        Invest      Withhold  
Farmer  
  Formal     3,2         -1,0  
  Informal   4,-1         0,0  
```  
**Justification**: AS5 in III.iv.a – Staff bear investment costs under formal cooperation, while informal requests create farmer/staff payoff asymmetry.

6. **Title**: Groundwater-Extraction Prisoner’s Dilemma  
**Tension**: Short-term individual incentives for over-extraction undermine long-term aquifer sustainability.  
**Matrix**:  
```
        Farmer B  
        Restrain    Over-Extract  
Farmer A  
  Restrain    3,3         1,4  
  Over-Extract 4,1         2,2  
```  
**Justification**: AS6 in III.iv.a – Mutual restraint preserves groundwater, but unilateral over-extraction yields higher payoffs, leading to depletion.
