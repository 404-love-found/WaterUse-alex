### Title: Distinct Action Situations in the Irrigation Electricity Governance Model

#### 1. Capacitor Adoption Assurance Game (AS1)
**Tension**: Farmer-Farmer Coordination
**Matrix Representation**:
```
               Farmer 2
               Invest     Not Invest
          Invest    (3, 3)       (0, 1)
Farmer 1  Not Invest (1, 0)       (2, 2)
```
**Justification**: This game captures the mutual coordination problem where both farmers benefit from investing in voltage-stabilizing equipment, but unilateral investment does not provide a private benefit, leading to a coordination dilemma.

#### 2. Sequential Social-Learning Process in Capacitor Adoption (AS2)
**Tension**: Social Learning and Coordination
**Sequential Representation (Game Tree)**:
```
Farmer 1
  \
   \
    (Invest) -> Farmer 2
         \
          \
           (Invest) -> (3, 3)
         / \
        /   \
    (Not Invest) -> (1, 1) (Not Invest) -> (1, 1)
```
**Justification**: This game represents the sequential process where each farmer observes the outcome of a peer’s investment and decides whether to invest based on the observed success. The tree structure captures the sequential decision-making process and the social learning mechanism.

#### 3. Asymmetric Transformer-Capacity Authorization Dilemma (AS3)
**Tension**: Free-Rider Incentive
**Matrix Representation**:
```
               Farmer 2
               Authorize     Not Authorize
          Authorize    (3, 3)         (1, 2)
Farmer 1  Not Authorize (2, 1)         (2, 2)
```
**Justification**: This game highlights the free-rider problem where mutual authorization benefits both farmers, but the cost falls solely on the authorizer, leading to an asymmetric incentive structure.

#### 4. Mutual-Exchange Coordination Game (AS4)
**Tension**: Informal Exchange and Reciprocity
**Matrix Representation**:
```
               Staff
               Exchange     No Exchange
          Exchange    (2, 2)         (1, 1)
Farmer 1  No Exchange (1, 1)         (3, 3)
```
**Justification**: This game captures the mutual exchange scenario where both the farmer and staff gain from informal exchange, but unilateral exchange results in a loss for the offerer and no gain for the recipient.

#### 5. Asymmetric Authorization-and-Investment Game (AS5)
**Tension**: Opportunism and Legality
**Matrix Representation**:
```
               Staff
               Formal Request  Informal Request
          Formal Request    (2, 2)         (1, 3)
Farmer 1  Informal Request (3, 1)         (3, 3)
```
**Justification**: This game represents the strategic tension between legal and informal requests, where mutual formal cooperation is optimal, but unilateral formal requests or informal investments lead to different outcomes with asymmetric incentives.

#### 6. Groundwater-Extraction Prisoner’s Dilemma (AS6)
**Tension**: Common-Pool Resource Extraction
**Matrix Representation**:
```
               Farmer 2
               Restrain     Over-Extract
          Restrain    (3, 3)         (1, 1)
Farmer 1  Over-Extract (1, 1)         (2, 2)
```
**Justification**: This game captures the dilemma where mutual restraint sustains groundwater yields, but unilateral over-extraction offers short-term gain at the cost of long-term depletion.
