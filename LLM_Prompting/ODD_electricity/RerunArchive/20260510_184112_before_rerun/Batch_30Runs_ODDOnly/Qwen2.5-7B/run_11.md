### Title: Distinct Action Situations in the Irrigation Electricity Governance Model

#### 1. Farmer-Farmer Capacitor Adoption Assurance Game (AS1)
**Tension:** Mutual Cooperation vs. Free-Rider Dilemma

**Matrix:**
```
                Farmer 2
                Invest   Not Invest
Farmer 1 Invest  (1, 1)    (0, 2)
      Not Invest (2, 0)    (0, 0)
```

**Justification:** This game represents the mutual benefit of both farmers investing in capacitors to improve voltage stability. However, if only one farmer invests, the other benefits without contributing to the cost, leading to a free-rider dilemma.

#### 2. Sequential Social-Learning Capacitor Adoption (AS2)
**Tension:** Diffusion of Best Practices vs. Conservative Behavior

**Sequential Representation (Game Tree):**
```
                Farmer 1
                  /   \
                 /     \
              Invest  Not Invest
                 |        |
                 v        v
       (1, 1)    (0, 1)
```

**Justification:** This represents a sequential learning process where a farmer observes the outcome of another farmer's capacitor investment and decides to invest or not based on the observed success. The game tree shows the sequential decision-making process, with the first farmer making the initial decision and the second farmer following based on the outcome.

#### 3. Asymmetric Transformer-Capacity Authorization Dilemma (AS3)
**Tension:** Farmer Investment vs. Staff Cost-Benefit Analysis

**Matrix:**
```
                Staff
                Authorize   Not Authorize
Farmer Invest  (1, -1)     (0, 0)
      Not Invest (0, 0)     (0, 0)
```

**Justification:** This game represents the dilemma where one farmer's investment to upgrade transformer capacity benefits both, but the costs fall entirely on the investing farmer. If the staff authorize the investment, the farmer benefits, but the staff bear the cost. If the farmer does not invest, both remain at a low baseline.

#### 4. Mutual-Exchange Coordination with Staff (AS4)


**Justification:** This game represents the mutual benefit of both parties engaging in informal exchanges. If both parties engage, they benefit, but if one party engages and the other does not, the engaging party bears the loss.

#### 5. Asymmetric Coordination Game with Staff (AS5)
**Tension:** Formal Cooperation vs. Opportunism

**Matrix:**
```
                Staff
                Formal   Informal
Farmer Invest  (1, 1)    (0, 1)
      Not Invest (1, 0)    (0, 0)
```

**Justification:** This game represents the dilemma where a farmer can request formal or informal authorization for transformer capacity. If both parties cooperate formally, they benefit, but if the farmer requests formally and the staff withhold, the farmer incurs a loss. If the farmer requests informally and the staff invest, the farmer benefits more.

#### 6. Groundwater Extraction Prisoner's Dilemma (AS6)

**Justification:** This game represents the mutual benefit of both farmers restraining groundwater extraction to sustain yields, but the temptation for one farmer to over-extract for short-term gain leads to a prisoner's dilemma.
