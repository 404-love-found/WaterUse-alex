### Action Situation 2: Sequential Social-Learning Process in Capacitor Adoption
#### Title: Farmer Social Learning
#### Tension: Imitation of successful peers in capacitor adoption
#### Matrix/Sequential Representation:
1. Farmer 1 adopts capacitor
2. Farmer 2 observes Farmer 1's outcome
   - If outcome is successful, Farmer 2 adopts capacitor
   - If outcome is not successful, Farmer 2 does not adopt capacitor
#### Justification: This sequential action situation represents the social-learning process among farmers, where each farmer observes a peer's outcome and imitates only if that outcome ranks higher.
