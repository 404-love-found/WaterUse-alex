### Action Situation 1: Capacitor Adoption Assurance Game (AS1)
**Tension:** Coordination to achieve mutual benefit vs. individual rationality

**Matrix Representation:**
```
                    Farmer 2
                    Invest    Not Invest
Farmer 1 Invest    (4,4)      (1,3)
Not Invest         (3,1)      (2,2)
```

**Justification:** This game represents the situation where two neighboring farmers must decide whether to invest in voltage-stabilizing equipment (capacitors). Mutual investment yields a shared improvement in electricity quality, but unilateral investment does not provide any additional private benefit. The game captures the coordination problem where mutual cooperation is Pareto-dominant but risky due to the potential for free-riding.

**Sequential Representation (Game Tree):**
```
Farmer 1
  \
   Farmer 2
  / \
 /   \

|     |

```

**Justification:** This represents the process where a farmer observes the outcome of another farmer’s capacitor adoption. The farmer imitates only if the observed outcome ranks higher. The game tree shows the sequential decision-making process where the first farmer makes the initial decision, and the second farmer observes and imitates if the outcome is better.

### Action Situation 3: Asymmetric Transformer-Capacity Authorization Dilemma (AS3)
**Tension:** Mutual benefit vs. free-rider problem

**Matrix Representation:**
```
                    Farmer 2
                    Authorize    Not Authorize
Farmer 1 Authorize    (3,3)        (5,1)
Not Authorize         (1,5)        (2,2)
```

**Justification:** This game models the decision of two farmers to authorize or not authorize transformer capacity. Authorizing benefits both but costs fall solely on the authorizer, creating a free-rider incentive. The game captures the dilemma where mutual cooperation is Pareto-dominant but risky due to the free-rider problem.

### Action Situation 4: Mutual-Exchange Coordination Game (AS4)
**Tension:** Formal cooperation vs. informal exchange

**Matrix Representation:**
```
                    Staff
                    Exchange   No Exchange
Farmer             (4,4)      (1,3)
                   No Exchange (3,1)      (2,2)
```

**Justification:** This game represents the interaction between a farmer and sub-station staff regarding informal exchanges of services. Mutual exchange yields reciprocal benefit, but unilateral exchange results in a loss for the offerer. The game captures the coordination problem where both formal and informal cooperation are possible but only matched cooperation yields mutual gain.

### Action Situation 5: Asymmetric Coordination Game (AS5)
**Tension:** Formal cooperation vs. opportunism

**Matrix Representation:**
```
                    Staff
                    Formal Invest   Formal Withhold
Farmer             (4,4)            (1,3)
                   Informal Invest (3,1)            (2,2)
```

**Justification:** This game models the interaction between a farmer and sub-station staff regarding the legality of capacitor investments. Formal cooperation is collectively optimal, but staff may withhold to save costs, and farmers may request informally to gain more. The game captures the asymmetric incentives between legality and opportunism.

### Action Situation 6: Groundwater Extraction Prisoner’s Dilemma (AS6)
**Tension:** Mutual restraint vs. short-term gain

**Matrix Representation:**
```
                    Farmer 2
                    Restrain    Overextract
Farmer 1 Restrain    (4,4)        (1,6)
Overextract          (6,1)        (2,2)
```

**Justification:** This game represents the situation where two farmers drawing from the same aquifer must decide whether to restrain or over-extract. Mutual restraint sustains yields, but over-extraction offers short-term gain and accelerates depletion. The game captures the prisoner’s dilemma where mutual restraint is Pareto-dominant but risky due to the potential for over-extraction.
